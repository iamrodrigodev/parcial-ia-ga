## Resumen Ejecutivo

Este proyecto implementa una aplicación web educativa para demostrar algoritmos genéticos aplicados a la resolución de laberintos. El sistema simula poblaciones de individuos (secuencias de movimientos), evalúa su aptitud y evoluciona generaciones mediante selección, cruce, mutación y elitismo.

La plataforma incluye dos módulos: (1) resolución de laberintos con visualización en tiempo real de la evolución y métricas, y (2) visualizadores didácticos de operadores genéticos (selección, cruce y mutación) para reforzar la comprensión conceptual.

Como resultado, se presenta una herramienta interactiva que combina explicación académica y experimentación práctica, permitiendo analizar el impacto de los parámetros evolutivos sobre convergencia, diversidad y calidad de soluciones.
